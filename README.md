# Password Generator
This program will generate a password with a user-defined length. 
Assignment provided by FreeCodeCamp.org

## Tools & Languages Used
* JavaScript

## Concepts Learned
* Do not place variables inside of functions that should exist on a global scale
* Scope

## Examples
```
JavaScript

const password = generatePassword(5);
console.log(`Generated password: ${password}`);
//Generated password: 49PF8

```

## How to Run
1. Clone the repository
2. Run the script using Node.js or import the function into any JS file
3. Change the paramater in generatePassword to any number
